Playground UX Mockups

Three product-grade directions for redesigning playground.wordpress.net, each inspired by a distinct real-world product.

Direction 1 Dark

Inspired by Figma

Floating toolbar, inspector panel, canvas metaphor. Settings in a right-side panel with tabs. Draft badge and dot-grid canvas. Great for users who think of their WordPress setup as a designable artifact.

Direction 2 Light

Inspired by Google Docs

Clean document-centered layout with autosave indicator, menu bar, toolbar ribbon, and blue Share button. Slash commands for plugin install. Settings in a right sidebar panel.

Open mockup

Direction 3 Dark

Inspired by VS Code

Activity bar, file tabs with unsaved dot, command palette, and status bar. Extensions panel for plugins. Settings as a tab. Keyboard-driven workflow with information-dense chrome.

Every mockup includes

Inline-editable title Click to rename the site/session. Enter to save, Esc to cancel.
Unsaved state indicator Visual feedback when settings change, styled per reference product.
Save / Export / Share Save to browser, download ZIP, push to GitHub, copy live URL.
Settings panel PHP/WP version, extensions, storage mode, network, language.
Plugin install Install plugins via the reference product's idiom (palette, slash, panel).
Keyboard shortcuts Ctrl/Cmd+S to save. Shortcut hints visible in all menus.
Responsive design Works at 360px, 768px, and 1280px. Panels collapse on mobile.
Feedback toasts Every action shows a toast notification with auto-dismiss.
